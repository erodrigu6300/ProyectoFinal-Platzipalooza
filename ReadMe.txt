Prerrequisitos: Contar con un servidor NodeJS.
1.	Ejecutar el siguiente comando: npm install -g serve
2.	Descargar el código fuente.
3.	Ubicarse en el código fuente.
4.	Ejecutar el siguiente comando: serve .

Una vez ejecutado los pasos descritos anteriormente, podrá visualizar la página web mediante el puerto configurado en su servidor; a continuación se relaciona una salida del comando descrito en el paso 4.
